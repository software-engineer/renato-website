Corso su tecnologie J2EE

L’obbiettivo del corso è stato quello di migliorare le conoscenze del mondo Web ed in modo particolare delle tecnologie J2EE. Durante lo svolgimento delle lezioni si sono realizzati diversi progetti di cui il principale è stata la realizzazione di una WebMail.
Tale web application è stata realizzata mediante l’utilizzo del framework Struts 1.3.
La connessione al data base MySQL veniva effettuata attraverso la libreria JDBC e le operazioni su di esso venivano elaborate attraverso il framework Hibernate.
L’application server utilizzato al fine di eseguire l’applicazione è stato JBoss. Al fine di permettere di migliorare le attività di gruppo, tale applicazione è stata realizzata mediante l’utilizzo di un SVN. Per la realizzazione della WebMail, inoltre, si è fatto uso delle librerie iText per la conversione delle email in pdf e delle librerie displaytag per la visualizzazione dei dati.
Oltre alle tecnologie sopracitate, in questo corso, si è perfezionato ed ampliato l’utilizzo di pagine HTML ed XHTML oltre che di JavaScript, JSP, Servlet.
Si è utilizzato anche il Servlet Container Tomcat con l’ausilio del Web Services AXIS.
Oltre al framework Struts 1.3, è stata studiato ed utilizzato anche il framework JSF.
Il corso è terminato con l’utilizzo degli EJB.

Facoltà di Ingegneria Informatica.

Titolo: Laurea in Ingegneria Informatica indirizzo Applicazioni in Rete.
Tesi: Location Based Services: Sviluppo di un’applicazione per il real-time tracking di dispositivi mobili di terza generazione.
Tecnologie e Linguaggi di Sviluppo: Java e C# con ambienti di sviluppo relativamente Eclipse e Visual Studio 2008.
Descrizione del lavoro svolto: Realizzazione di un sistema client server.
In questo sistema il client, dispositivo mobile dotato di sistema gps, invia periodicamente, al server, le proprie coordinate di posizionamento. Il server, riceve le coordinate dei dispositivi ad esso associato, le memorizza e attraverso il servizio google map, permette di visualizzare lo storico delle posizioni del dispositivo su mappe google. In modo particolare, è possibile visualizzare l’intero percorso svolto dal dispositivo mobile.
